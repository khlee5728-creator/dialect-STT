사투리 특화 AI 말동무 서비스

어르신 대상 방언 인식 AI 케어 플랫폼

실버산업 AI 적용 프로젝트

🎯 프로젝트 전략 방향

국내 유일 사투리 특화 AI 말동무 서비스

기존 노인 돌봄 AI가 해결하지 못한 두 가지 문제를 동시에 해결합니다.
  • 언어 장벽: 표준어 중심 AI는 사투리를 쓰는 어르신과 제대로 대화하지 못함
  • 관계 단절: 주 1회 3분짜리 안부 확인은 진짜 말동무가 될 수 없음

사투리 STT/TTS 기술을 자체 개발·고도화하며, 서비스 운영을 통해 축적되는 실사용 데이터가 기술 경쟁력을 지속적으로 강화합니다. 장기적으로는 사투리 음성 AI 기술을 디바이스·플랫폼 파트너에게 공급하는 기술 기반 서비스 회사로 성장합니다.

전략 로드맵

1단계 — 서비스 출시·검증

사투리 STT 자체 개발 (경상도 우선)로 기술 차별화 확보
전화 기반 AI 말동무 서비스 출시
어르신·가족·지자체 레퍼런스 확보
실사용 대화 데이터 축적 → 기술 고도화 재료

2단계 — 동시 고도화 (6개월~1년)

실사용 데이터 기반 사투리 STT 전 권역 확장
가족 앱 출시
디바이스 파트너십 협의 시작

3단계 — 기술 사업화 (장기)

사투리 음성 AI API 외부 공급 (효돌㈜ 등)
지자체 화이트라벨 납품
특허 기반 기술 라이선싱

📄 서비스 개요

실버산업 AI 적용 프로젝트로, 어르신 대상 AI 말동무 서비스에 사투리 STT/TTS를 접목해 기존 서비스와 차별화된 노인 케어 플랫폼을 개발합니다.

⭐ 핵심 차별화 포인트

장기 기억

대화 히스토리를 누적하여 어르신의 인생사와 관심사를 기억하고 맥락 있는 대화를 이어갑니다.

사투리 이해

경상 / 전라 / 충청 / 제주 방언을 인식하고 해당 방언으로 자연스럽게 응답합니다.

가족 연결 3자 구조

어르신 ↔ AI ↔ 가족 앱 연동으로 보호자가 대화 상태를 확인할 수 있습니다.

디지털 회고록

대화 기반으로 어르신의 인생 기록을 자동 생성하여 가족에게 전달합니다.

⚙ 기술 스택

STT (Speech-to-Text)

OpenAI Whisper + Deepgram
  • Whisper: 파인튜닝 가능 오픈소스 (방언 특화)
  • Deepgram: 8kHz 전화 음질 최적화, 실시간 스트리밍
  • AI Hub 중·노년층 한국어 방언 데이터 활용
  • 경상도·충청도·전라도·제주도 방언

전략: 범용 STT + 사투리 후처리 정규화 레이어 (단기) → Whisper 파인튜닝 (중기)

LLM

Claude API (claude-haiku)
  • 빠른 응답 우선 (~800ms)
  • 사투리 표현 few-shot 예시 포함
  • 어르신 말투·관심사 컨텍스트 주입
  • 장기 기억 요약 포함

TTS (Text-to-Speech)

Clova Voice + 사투리 어미 치환
  • 경상도: ~했심더, ~이라예, ~했능교
  • 전라도: ~했제, ~이여, ~했당께
  • 충청도: ~했슈, ~이유, ~했쥬

중기: 사투리 어미 특화 TTS 파인튜닝
장기: 노인 음성 특징(느린 속도·낮은 음역) 반영 엔드투엔드 모델

인프라

Twilio 양방향 전화 + FastAPI
  • Twilio Outbound/Inbound Call API
  • APScheduler 기반 발신 스케줄러
  • FastAPI + WebSocket 서버
  • PostgreSQL (대화 요약·감정 저장)
  • React Native (가족 앱)

💬 AI 대화 기획

페르소나 설정

캐릭터친근한 손녀·손자 같은 젊은이
말투사투리 권역별 맞춤 + 존댓말 기반의 따뜻하고 편안한 어조
이름지역별 친근한 이름 설정 가능 (예: "하늘이", "봄이") — 지자체 화이트라벨 납품 시 커스터마이징 가능
태도 · 항상 먼저 안부를 묻고 어르신 말을 충분히 들음
· 절대 서두르지 않음 — 어르신 발화 속도에 맞춤
· 어르신이 틀린 말을 해도 교정하지 않음
· 걱정·불안 표현 시 공감 먼저, 해결책 나중
금지 행동 · "잠깐만요", "빠르게 말씀드리면" 등 재촉하는 표현 금지
· 너무 많은 질문을 한꺼번에 던지지 않음 (한 번에 하나씩)
· 어르신이 반복되는 이야기를 해도 처음 듣는 것처럼 반응

대화 구조 (하루 1회 / 10~15분)

1단계 — 오프닝 (1~2분)

AI가 먼저 전화 → 인사 + 날씨·계절 안부
"어르신, 저 하늘이예요~ 오늘 날씨가 많이 쌀쌀하던데 따뜻하게 계셨어예?"

2단계 — 감정 체크 (2~3분)

어르신 기분·컨디션 파악, 어제 대화 기억 기반으로 자연스럽게 연결
"어제 무릎이 좀 아프시다고 하셨는데, 오늘은 좀 어떠세예?"

3단계 — 메인 대화 (5~8분)

그날의 콘텐츠 테마에 따라 진행, 어르신이 주도하면 자연스럽게 따라감

4단계 — 건강·복약 체크 (1~2분)

식사 여부, 복약 확인을 자연스럽게 대화 속에 녹임
"오늘 점심은 맛있는 거 드셨어예? 약도 챙겨 드셨죠?"

5단계 — 클로징 (1분)

가족 근황 한 마디 전달 + 다음 통화 예고
"아드님이 이번 주말에 찾아뵌다고 했어예~ 내일 또 전화드릴게예!"

대화 콘텐츠 테마 (요일별 순환)

요일테마예시 대화 소재
월일상 안부·감정 케어주말 어떻게 보내셨어요, 요즘 기분은요
화인생 회상젊었을 때 직업·고향·첫사랑 이야기
수인지 훈련옛날 노래 맞히기, 계절 퀴즈, 숫자 기억
목가족 근황 연결자녀·손자녀 소식 전달, 가족 사진 이야기
금건강·일상 관리식사, 수면, 산책, 복약 점검
토인생 회고록 기록이번 주 있었던 일 정리, 기억에 남는 순간
일자유 대화어르신이 하고 싶은 이야기 주도

콘텐츠별 세부 설계

① 일상 안부·감정 케어

매일 오프닝·클로징 기본 포함. 직접 "기분이 어때요?" 대신 상황으로 유도. 부정적 감정 감지 시 공감 대화 → 감정 완화 후 주제 전환. 위기 표현 감지 시 알림 시스템 연동.

② 인생 회상·회고록 기록

고향 → 어린 시절 → 학창시절 → 직업 → 결혼 → 자녀 양육 → 기억에 남는 순간 → 인생의 지혜. 자동 텍스트 저장 → 회고록 누적, 주간 가족 리포트 발송.

③ 인지 훈련 (치매 예방)

게임처럼 자연스럽게 — 테스트 느낌 금지. 회상 훈련, 언어 훈련(노래·속담), 지남력 훈련(날짜·계절), 주의력 훈련(끝말잇기). 인지 반응 패턴 DB 저장 → 장기 추이 모니터링.

④ 건강·복약 관리

대화 중 자연스럽게 체크. 식사·수면·통증·복약·외출 점검. 이상 패턴 감지 시 가족 알림 (예: "3일 연속 밥을 못 드셨다").

⑤ 가족 근황 연결

가족 앱에서 보호자가 입력한 소식을 AI가 대화 중 자연스럽게 전달. 어르신이 가족에게 전하고 싶은 말을 AI가 받아서 가족 앱으로 전달.

장기 기억 구조

기본 프로필

이름, 나이, 고향, 방언 권역, 가족 구성원
좋아하는 것 (음식, 노래, TV)
건강 상태, 복약 목록, 병원 방문 주기

대화 기억

최근 7일 대화 요약 (LLM 자동 생성)
주요 에피소드 태그 (손녀 결혼식, 무릎 통증)
감정 상태 추이 (weekly)

회고록 데이터

인생 스토리 누적 텍스트
주제별 정리 (고향, 직업, 가족, 기억에 남는 순간)

LLM 프롬프트에 매 대화마다 기본 프로필 + 최근 7일 요약 주입 → "지난번에 말씀하셨는데…" 같은 자연스러운 연속성 확보

대화 예시 (경상도 어르신)

AI: "어르신~ 저 하늘이예요! 오늘 많이 추웠지예? 어제 무릎 아프시다 하셨는데 좀 나아지셨어예?"

어르신: "아이고 하늘아, 오늘은 좀 낫다. 근데 날이 추우니께 또 쑤시는 기라."

AI: "그라제예~ 추우면 더 심하지예. 따뜻하게 핫팩이라도 붙이고 계시지예? 오늘 점심은 뭐 드셨어예?"

어르신: "국 끓여 먹었다. 혼자 먹으니께 맛도 없고..."

AI: "아이고, 혼자 드시니 좀 쓸쓸하시지예. 그라도 어르신 국 솜씨가 어디 가겠어예~ 젊었을 때는 요리 잘하셨다 하지 않으셨어예?"

대화 품질 관리 원칙

  • 한 번에 질문 하나: 여러 질문 동시 금지
  • 침묵 허용: 어르신이 생각할 시간 충분히 줌 (3초 이상 기다림)
  • 반복 수용: 같은 이야기 반복해도 처음 듣는 것처럼 반응
  • 과도한 긍정 금지: "너무 좋아요!", "대단해요!" 남발 금지
  • 의료 조언 금지: 증상 언급 시 "병원에 가보시는 게 좋을 것 같아예" 수준으로 제한
  • 강요 금지: 대화하기 싫어하시면 "그러시면 내일 또 전화드릴게예" 하고 종료

알림 메시지 예시

① 미수신 알림

[AI 말동무] 홍길동 어르신께 오전 10시, 10시 15분 두 차례 전화를 드렸으나 받지 않으셨어요. 직접 확인해 보시겠어요?

② 감정 이상 감지

[AI 말동무] 오늘 홍길동 어르신과의 대화에서 평소와 다른 표현이 감지되었어요. 대화 요약 보기 → [링크]

③ 일일 리포트

[AI 말동무] 오늘 홍길동 어르신 대화 요약 — 통화시간: 12분 | 기분: 좋음
"손녀 결혼식 얘기를 많이 하셨어요"

④ 장기 미응답 알림

[AI 말동무] 홍길동 어르신과 3일째 대화가 이루어지지 않고 있어요. 안부 확인이 필요할 수 있습니다.
