사투리 특화 AI 말동무 서비스

어르신 대상 방언 인식 AI 케어 플랫폼

실버산업 AI 적용 프로젝트

📄 프로젝트 개요

실버산업 AI 적용 프로젝트로, 어르신 대상 AI 말동무 서비스에 사투리 STT/TTS를 접목해 기존 서비스와 차별화된 노인 케어 플랫폼을 개발합니다.

⭐ 핵심 차별화 포인트

장기 기억

대화 히스토리를 누적하여 어르신의 인생사와 관심사를 기억하고 맥락 있는 대화를 이어갑니다.

사투리 이해

경상 / 전라 / 충청 / 제주 방언을 인식하고 해당 방언으로 자연스럽게 응답합니다.

가족 연결 3자 구조

어르신 ↔ AI ↔ 가족 앱 연동으로 보호자가 대화 상태를 확인할 수 있습니다.

디지털 회고록

대화 기반으로 어르신의 인생 기록을 자동 생성하여 가족에게 전달합니다.

초저마찰 인터페이스

전화 한 통으로 시작 — 앱 설치나 스마트폰이 필요 없습니다.

⚙ 기술 스택

STT (Speech-to-Text)

OpenAI Whisper
  • 파인튜닝 가능 오픈소스 모델
  • AI Hub 중·노년층 한국어 방언 데이터 활용
  • 경상도 방언 (50대 이상 발화자)
  • 충청도·전라도·제주도 방언

전략: 범용 STT + 사투리 후처리 정규화 레이어 (단기) → Whisper 파인튜닝 (중기)

LLM

Claude API (claude-sonnet)
  • 사투리 표현 few-shot 예시 포함
  • 어르신 말투·관심사 컨텍스트 주입
  • 장기 기억 요약 포함

TTS (Text-to-Speech)

표준어 TTS + 사투리 어미 치환
  • 경상도: ~했심더, ~이라예
  • 전라도: ~했제, ~이여
  • 충청도: ~했슈, ~이유

중기: 사투리 어미 특화 TTS 파인튜닝
장기: 노인 음성 특징(느린 속도·낮은 음역) 반영 엔드투엔드 모델

인프라

멀티 인터페이스
  • React Native (모바일 앱 — 가족용)
  • 음성 전화 인터페이스 (어르신용)
  • 카카오톡 연동 검토

🚀 사투리 STT 구현 전략 (단계별)

1단계 — MVP (빠른 검증)

범용 Whisper → 사투리 후처리 사전(dict) → LLM → 표준어 TTS
어미만 사투리풍으로 조정하여 최소 기능 검증

2단계 — 차별화

AI Hub 방언 데이터로 Whisper 파인튜닝
경상도 우선 → 전라도 → 충청도 순으로 확장

3단계 — 풀스택

노인 음성 특화 STT (느린 발화·약한 발음 대응)
사투리 TTS 파인튜닝 / 특허 출원
